# Лендинг для разработчика

Одностраничный сайт для привлечения заказов: боты, сайты, мини-приложения, автоматизация.

## Как открыть

1. Откройте `index.html` в браузере (двойной клик или через Live Server в VS Code).
2. Или поднимите локальный сервер:
   ```bash
   cd landing
   python -m http.server 8080
   ```
   Сайт: http://localhost:8080

## Что подставить под себя

В `index.html` замените:

- **Telegram:** `https://t.me/your_username` → ваш ник в Telegram.
- **Email:** `your@email.com` в ссылке `mailto:your@email.com`.

В блоке «Примеры работ» замените заглушки на реальные кейсы (название + ссылка или скрин).

## Форма заявок

Сейчас при отправке формы данные выводятся в консоль браузера (F12 → Console). Чтобы получать заявки:

1. **Вариант 1.** Подключите форму к Telegram-боту (отправка в личку или в канал через Bot API).
2. **Вариант 2.** Сделайте бэкенд (например, на Flask в папке `ad/`), который принимает POST и шлёт уведомление вам в Telegram или на почту.

Файлы: `index.html`, `styles.css`, `script.js`.
